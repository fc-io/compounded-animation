## Compounded Animation

An animation control flow experiment

### Getting Started

1. `git clone https://github.com/fc-io/react-tape.git`
2. `cd react-tape`
3. `npm install`
4. `npm start`
5. open [http://localhost:3000](http://localhost:3000)

### Build for Production

```
npm run deploy
```

### Testing

```
npm test
```

### Linting

```
npm run lint
```
